# SQL Practice Project
Этот проект представляет собой коллекцию SQL-скриптов, демонстрирующих создание таблиц, 
вставку данных, использование агрегатных функций, фильтрацию, объединение таблиц и другие операции с базой данных. 

 Содержание файлов

1. Create_table.sql

Создаёт таблицу rooms с колонками:
- room_id — ID комнаты;
- teacher_id — ID преподавателя;
- floor_num — номер этажа.

2. HW Основные типы SQL-запросов.sql

Набор выборок из таблиц student, teacher, users:
- Поиск уровня студента;
- Электронные адреса преподавателей;
- Фильтрация пользователей по почтовым доменам.

3. SQL-запросы квест.sql

Разнообразные запросы к таблице species и связанным таблицам:
- Фильтрация по статусам, количеству, дате, имени;
- Используются JOIN, ORDER BY, LIKE, LIMIT, OFFSET.

4. hw2.sql
   
Запросы с соединением таблиц:
- group_student, teacher, student, users;
- Подсчёт значений, фильтрация, агрегации.

5. Агрегация данных практика.sql
   
Работа с таблицей sales:
- Поиск максимальной, минимальной и средней суммы;
- Подсчёт уникальных покупателей.

6. Агрегация данных.sql
   
Большая подборка запросов с агрегацией и группировкой:
- Подсчёты по уровням, предметам, формам обучения;
- Использование GROUP BY, HAVING, ORDER BY.

7. ДЗ агрегация.sql
   
- Подсчёт студентов по условиям;
- Использование GROUP BY, ORDER BY, LIMIT, MIN, MAX.

8. Добавление записей в табл.sql
   
Вставка новых строк в таблицу rooms с указанием ID преподавателей и этажей.

10. Соединение таблиц hard.sql
    
JOIN-запросы:
- Фильтрация по чекам;
- Поиск непокупаемых товаров;
- Подсчёт чеков по условиям по категориям и количеству товаров.

10. Соединение таблиц.sql
    
Запросы с различными типами соединений:
- INNER, LEFT, RIGHT, FULL, CROSS JOIN;
- Связи между пользователями, предметами, группами и преподавателями.

11. работа с бд (операторы, изменение табл).sql
    
Примеры работы с данными:
- Фильтрация постов;
- Вставка и обновление записей в таблицах posts и tags.
